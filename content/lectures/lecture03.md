# Lecture 3 — <Основы CSS>
**Тема:** <стилизация веб-страниц>
**Цель урока:** (научиться оформлять страницы)
**Время:** ~20 мин.
**Ключевые понятия:** <CSS, селекторы, свойства, цвета>

## 1) Введение
CSS отвечает за внешний вид страницы: цвета, размеры, расположение элементов.

## 2) Основная часть
• Подключение CSS:
inline, <style>, отдельный файл
• Селекторы:
h1, .class, #id
• Основные свойства:
color, background, font-size
• Блочная модель:
margin, border, padding
• Простое позиционирование:
display, flex

## 3) Пример/практика
body {
  background-color: lightgray;
}

h1 {
  color: blue;
  margin: 20px;
}
