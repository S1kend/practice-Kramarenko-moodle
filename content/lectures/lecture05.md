## 1) Введение
Git — система контроля версий, помогает отслеживать изменения и работать в команде.

## 2) Основная часть
• Репозиторий: место хранения проекта
• Основные команды:
git init — создать репозиторий
git add — добавить файлы
git commit — сохранить изменения
• История изменений:
возможность отката
• GitHub:
хранение проектов онлайн
• Push/Pull:
отправка и получение изменений

## 3) Пример/практика
git init
git add index.html
git commit -m "add index page"
