# Lecture 2 — <Основы HTML>
**Тема:** <структура веб-страницы>
**Цель урока:** (научиться создавать простую HTML-страницу)
**Время:** ~20 мин.
**Ключевые понятия:** <HTML, теги, структура документа>

## 1) Введение
HTML описывает структуру страницы: что является заголовком, текстом, ссылкой и т.д.

## 2) Основная часть
• Структура документа:
<html> — корень
<head> — метаданные
<body> — содержимое
• Основные теги:
<h1> — заголовок
<p> — текст
<a> — ссылка
<img> — изображение
• Атрибуты:
href, src, alt
• Семантика:
<header>, <main>, <footer> — улучшают структуру

## 3) Пример/практика
<!DOCTYPE html>
<html>
<head>
  <title>Моя страница</title>
</head>
<body>
  <h1>Привет!</h1>
  <p>Это мой первый сайт</p>
  <a href="https://example.com">Ссылка</a>
</body>
</html>
