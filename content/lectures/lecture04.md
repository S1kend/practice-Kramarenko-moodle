## 1) Введение
JavaScript добавляет интерактивность: кнопки, формы, динамика.

## 2) Основная часть
• Подключение: <script>
• Переменные:
let, const
• Типы данных:
число, строка, boolean
• Функции:
повторное использование кода
• События:
onclick, input
• Работа с DOM:
изменение текста и элементов

## 3) Пример/практика
<button onclick="sayHello()">Нажми</button>

<script>
function sayHello() {
  alert("Привет!");
}
</script>
