## 1) Введение
Веб-технологии — это инструменты для создания сайтов и веб-приложений. Они позволяют пользователю взаимодействовать с информацией через браузер.

## 2) Основная часть
• Интернет vs Веб:
Интернет — это сеть, веб — сервис внутри неё
• Клиент–сервер:
клиент (браузер) отправляет запрос → сервер отвечает
• HTTP/HTTPS:
протокол передачи данных (безопасная версия — HTTPS)
• Как загружается сайт:
a. Ввод URL
b. DNS → IP
c. HTTP-запрос
d. Ответ сервера
e. Рендеринг страницы

## 3) Пример/практика
Открыть DevTools → вкладка Network → обновить страницу → посмотреть запросы (HTML, CSS, JS).
